

## 需要给游戏端开启的方法 ##

进入游戏进度条 进度值  0-100
```
progress(value: number)  
getProgress(value: number)  # 优选调用progress
```

通知进入游戏了
````
gameOnload()
````
打开指定的web页面 不关闭游戏的前提下  value 路径 如： "/giftPage?token=***"
````
openWebPageWithoutLeaveGame(value: string)
````
打开大厅或原生的默认弹窗  value 提示文案
````
openModal(value: string)
````
关闭游戏 type: 0.默认直接退出 1.退出切换到新游戏 data:附带其它数据
````
gameClose(type = 0, data = null)
````
 进入充值界面
````
deposit()
````
 进入登录界面
````
login()
````
 进行用户上传头像
````
uploadAvatar() #优先
updateHead()
````
